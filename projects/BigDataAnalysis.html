Big Data Analysis Using Hadoop
The Project involves Analysis over 3 DataSets and its graphical representation along with its summary.
DataSets:

Forum DataSet:

The dataset was taken from the Udacity forums.
The basic structure is- the forum has nodes, All nodes have a body and author_id.
Top level nodes are called questions, and will also have a title and tags.
Questions can have answers. Both questions and answers can have comments
Size of almost 1.1 gb.
It involves following problems and solutions.
1)Top Tags : It shows domain wise discussion overview.
2)Student Times: It allows you to see how many users are active online and which is the suitabe time to post any question.

Purchases DataSet:

The dataset consist of various purchases records of number of stores of a particular firm.
It includes Date, time and category of purchased product along with information about store and cost of product and having a size of 985 Mb.
It solves following problems.
1)Sales Per Category: It shows category-wise sale of all the products and its amount.
2)Sales Per City: It shows you city-wise sale scenario of the stores located there.
3)Weekday Sale: It shows you day-wise sale scenario for whole organization.

Access Log DataSet:

The data set used is an anonymized Web server log file from a public relations company whose clients were DVD distributors. Each line in dataset represents a hit to the Web server.
It includes the IP address which accessed, the site, the date and time of the access, and the name of the page which was visited.
It involves the following task:
Determine the number of hits to the site made by top 9 different IP address

This was All about static data analysis . Real time analysis can also be performed by clicking over the button in sidebarpanel.

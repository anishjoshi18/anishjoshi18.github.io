Sentimental Analysis Tool
The Project involves Data mining of the real time data which is fetched from twitter and some websites.
This project is a live website which anyone can use.
How to use it--(With description about its back-end)
1. Enter a Valid Query (Make sure that u haven't committed a spelling mistake and it must be a relevant)
2. Select no of tweets on which you have to perform the analysis.
3. Select Genre of your query
a) Movie--(All the movie related sites will be linked along with the data fetched from the twitter)
b)News--(All the news related sites will be linked along with the data fetched from the twitter)
c)Sports--(All the sports related sites will be linked along with the data fetched from the twitter)
d)World--(Tweets from the whole world are fetched)
e)India--(Tweets only from India are fetched)
4. After all this click on Analyse button
**ANALYSIS**
WordCloud---WordCloud related to the query you have entered is seen.
Sentimental Graph--It shows various emotions such as joy,sad,anger,fear,disgust,surprise etc linked with your entered query.
Positive-negative Graph-A graph build with the help of Naive Bayes Classifier is shown .
Trending-Tweets--Trending tweets on Twitter.
Data--You can see all the data on which the analysis has been performed.
You can even download all the analysis you have made :)
